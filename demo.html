WGS Demo

WGS is a library allowing the generating of a set of OOCSS classes. Theses classes are heavily oriented toward the creation responsive webpage structures.

Source code available on github

// Sample configuration @import "wgs"; @include wgsSetup( $breakpoints : ( tablet : 480px, big-tablet : 660px, desk : 990px ), $gutters : 12px 16px 18px 22px, $fixedWidths : 200px 300px );

What does this sample config mean:

  • Enable the use of three named breakpoints. The first one starts at 480px, and is named "tablet". The second one starts at 660px and is named "tablet". Guess the third one :)
  • Enable the use of gutters widths and spacings classes, with breakpoint sufixes: .grid--12px, .grid--22px@desk, .padding--12px, .padding-top--22px@desk...
  • Enable the use of 200px and 300px fixed widths. We can use .200px and .300px classes, with sufixes provided in $breakpoints parameter: .200px@tablet, 300px@desk...
