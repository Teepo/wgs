    @mixin wgsSetup($breakpoints: (), $gutters: (), $fixedWidths: (), $mobileFirst: ()) { ... }

    Description

    Main function. has to be called with wanted fixed widths, breakpoints definition and gutter widths to handle

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $breakpoints

    breakpoints

    		Map()
    $gutters

    gutters to handle

    		List()
    $fixedWidths

    fixed sizes to handle

    		List()
    $mobileFirst

    mobile first ? (or desktop first)

    		Boolean()

    Example

      $breakpoints : (
          tablet     : 480px,
          big-tablet : 660px,
          desk       : 990px
      ),

      $fixedWidths : 200px 300px
  );
